What is culture?

Stories are contagious.

If technology is practical application of science, then culture is practical application of art.

Various art, with various embedded stories, can be used to create various kinds of cultures, which (across varying timeframes and scales) can produce various effects.

The faster the world moves, the more volatile culture becomes, and the more it pays to understand these phenomena from principles.

Which begs the question: which principles?
